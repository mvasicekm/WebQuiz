Sudost a lichost funkce
Otázka Sudost funkce ◄ Otázky ►
Funkce f se nazývá sudá, právě když zároveň platí:

  1. Pro každé x ∈Df je také − x ∈Df,

  2. Pro každé x ∈Df je f(−x) = f(x).

Funkce f se nazývá lichá, právě když zároveň platí:

  1. Pro každé x ∈Df je také − x ∈Df,

  2. Pro každé x ∈Df je f(−x) = −f(x).

Která z nabízených funkcí je zároveň sudá i lichá?
(a)
f(x) = 2x
(b)
f(x) = 0
(c)
f(x) = 4
(d)
Taková funkce neexistuje

Volba (a) je Špatně!
sudost: f(x) = 2x,f(−x) = −2x ⇒ není sudá, protože 2x≠ − 2x
lichost: f(x) = 2x,−f(−x) = −(−2x) = 2x ⇒ je lichá, protože 2x = 2x.
Volba (b) je Dobře!
Správně!
Funkce je sudá, protože f(x) = 0,f(−x) = 0 a lichá, protože f(x) = 0,−f(−x) = −(0) = 0.
Volba (c) je Špatně!
sudost: f(x) = 4,f(−x) = 4 ⇒ je sudá, protože 4 = 4
lichost: f(x) = 4,−f(−x) = −(4) = −4 ⇒ není lichá, protože 4≠ − 4.
Volba (d) je Špatně!
Ale existuje. Zkus to znovu.
